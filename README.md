# StripeRight Landing Page (React + TypeScript)

A modern landing page for StripeRight built with **React + TypeScript + Vite**.

## Services highlighted

- Stripe painting
- Concrete sealing
- Pressure washing

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal (typically `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
